## Usage

### Requirements
* [Node.js](http://nodejs.org/) (required for npm & Grunt)
* [npm](https://npmjs.org/)
* [Grunt CLI](http://gruntjs.com/getting-started)

### Build instructions

* `npm install`
* `grunt --target=[all, win, mac, linux32, linux64]`
